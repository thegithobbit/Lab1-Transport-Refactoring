# Глосарій предметної галузі
* **Транспорт (Transport)** — базовий клас, що визначає загальні характеристики (назва, номер, швидкість).
* **Наземний транспорт (LandTransport)** — спеціалізація для засобів, що мають колеса та пасажиромісткість.
* **Черга (Queue)** — структура даних для зберігання об'єктів транспорту.
* **AVL-черга** — черга, що використовує самозбалансоване дерево для швидкого пошуку за пріоритетом.
